# Admin Service - Food Delivery Platform

Spring Boot microservice for food delivery platform administration.

## Features
- User management
- Restaurant management
- Delivery personnel management
- JWT authentication
- PostgreSQL database
- RESTful API

## Tech Stack
- Java 17
- Spring Boot 4.0.1
- PostgreSQL
- JWT (JJWT)
- Lombok
- Maven

## Getting Started

### Prerequisites
- Java 21+
- PostgreSQL 13+
- Maven 3.6+

### Installation
```bash
git clone https://github.com/YOUR_USERNAME/admin-service.git
cd admin-service
mvn clean install
